# PondsiHotspot
一个用于在Windows 10+开启热点的脚本

## pondsihotspot.ps1

一个可以开启/关闭电脑 WiFi 热点的 PowerShell 脚本
需要以管理员权限运行
__参数示例：__
_`-EnableHotspot`: 开启WiFi热点。_
_`-DisableHotspot`: 关闭WiFi热点。_
_`-EnableAdapter`: 启用无线网络适配器。_
_`-DisableAdapter`: 禁用无线网络适配器。_
_`-Force`: 忽略管理员权限检测运行脚本。_
_`-help`: 获取帮助_
_如果无输入参数则自动开/关热点_
